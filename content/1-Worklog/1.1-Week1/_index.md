### Week 1 Objectives:
* Register for an AWS Free Tier account to prepare the practice environment.
* Understand budget management and cost monitoring to avoid unexpected billing.
* Learn about AWS Support to handle technical issues if they arise.
* Master the basics of identity management and account security using AWS IAM.

### Week 1 Achievements:
* **AWS Account Creation:** Successfully registered a Free Tier account, linked a payment card, and verified identity.
* **AWS Budgets:** Created budget alarms when costs reach 80% and 100% of a self-imposed limit (e.g., $5/month) to avoid unexpected billing.
* **AWS Support:** Understood the difference between Basic, Developer, Business, and Enterprise support tiers. Learned how to create support cases for technical issues.
* **AWS IAM:**
  - Avoided using the Root account for daily tasks following security best practices.
  - Created user groups such as *Admins* and *Developers* and assigned permissions using Policies.
  - Enabled MFA for the Root account and critical IAM Users.
  - Learned the structure of a JSON Policy document, including: Effect (Allow/Deny), Action, Resource, and Condition.
