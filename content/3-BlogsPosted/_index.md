###  [Blog 1 - AWS SYSTEMS MANAGER – CENTRALIZED MANAGEMENT SOLUTION FOR HYBRID CLOUD ENVIRONMENTS](3.1-Blog1/)
This blog introduces AWS Systems Manager (SSM) - a service that helps centrally manage resources on AWS and On-Premises servers from a single platform. With features like Session Manager (secure server access without SSH/RDP) and Patch Manager (automated system updates), this is a powerful solution that helps businesses manage Hybrid Cloud environments in a centralized, secure, and efficient manner.

###  [Blog 2 - AWS AGENTIC AI – INTELLIGENT AI APPLICATION IN SAP ERP SYSTEMS](3.2-Blog2/)
This blog introduces Agentic AI - a solution that applies Generative AI to SAP ERP systems to support Exception Management. Unlike traditional chatbots, Agentic AI can analyze data, reason, and propose solutions by combining Amazon Bedrock, Bedrock AgentCore, and other AWS services, opening up a new approach to optimizing business processes.

###  [Blog 3 - AWS SECURITY ANALYTICS – BUILDING CLOUD SIEM SYSTEM WITH AMAZON OPENSEARCH AND SECURITY LAKE](3.3-Blog3/)
This blog introduces a fully cloud-based Security Analytics solution by combining Amazon Security Lake with Amazon OpenSearch. This architecture automatically standardizes security log data to OCSF format, detects threats in near real-time, and supports Zero-ETL, helping build a robust SIEM (Security Information and Event Management) system capable of processing Terabytes of logs daily without maintaining physical infrastructure.
