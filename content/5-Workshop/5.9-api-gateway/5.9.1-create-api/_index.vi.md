#### Các bước thực hiện

**1. Truy cập API Gateway Console**

- Tìm kiếm **API Gateway**
- Click vào **API Gateway**

**2. Create API**

- Click **Create API**
- Chọn **REST API** (không phải REST API Private)
- Click **Build**

**3. API Settings**

- Chọn **New API**
- API name: `itcoach-api`
- Description: `ITCoach REST API for frontend`
- Endpoint type: **Regional**

![Create API](/images/5-Workshop/5.9-api/001-create-api.png)

- Click **Create API**

#### Kết quả

✅ REST API `itcoach-api` đã được tạo với root resource `/`
