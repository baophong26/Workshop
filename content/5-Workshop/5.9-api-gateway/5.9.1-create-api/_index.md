#### Steps

**1. Access API Gateway Console**

- Search for **API Gateway**
- Click on **API Gateway**

**2. Create API**

- Click **Create API**
- Select **REST API** (not REST API Private)
- Click **Build**

**3. API Settings**

- Select **New API**
- API name: `itcoach-api`
- Description: `ITCoach REST API for frontend`
- Endpoint type: **Regional**

![Create API](/images/5-Workshop/5.9-api/001-create-api.png)

- Click **Create API**

#### Result

✅ REST API `itcoach-api` created with root resource `/`
